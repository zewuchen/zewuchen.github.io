Amianto

Jogo criado com o objetivo de incluir pessoas com deficiência visual em jogos. Este jogo tem como proposta encontrar as travas dispostas na tela com o intuito de desativar uma bomba para que o jogador continue participando do jogo, com um número limitado de até 4 jogadores o jogo seleciona aleatoriamente um dos jogadores por meio de uma simulação de ligação dentro do jogo para que o usuário tenha sua participação. Todos as instruções são feitas por meio de efeitos sonoros, de haptic feedback e jogo adaptado para ser usado com o recurso de acessibilidade do iOS VoiceOver, sendo assim, não necessário o uso visual para poder ser jogado. Trabalhei na formulação das telas, mapeamento dos dedos do usuário para localização das travas e uso de haptic feedback. Sua construção foi feita por meio de Storyboard e em UIKit. Este projeto teve duração de dois meses
